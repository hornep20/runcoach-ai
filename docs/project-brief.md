# RunCoach AI

RunCoach AI is a full-stack marathon coaching web app that integrates with Intervals.icu to pull COROS training data, display training dashboards, manage base-building and marathon training plans, and provide personalized AI coaching using RAG.

Core features:

- Intervals.icu sync
- Running dashboard
- Training calendar
- Base-building plan builder
- 16-week marathon plan builder
- Strength and mobility workout generator
- AI coach with RAG
- PDF training plan upload later

Tech stack:

- Next.js
- TypeScript
- Tailwind
- shadcn/ui
- PostgreSQL
- Prisma
- pgvector
- OpenAI API
